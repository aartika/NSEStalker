NSEStalker by soumendra

Tools for downloading data from NSE website and managing and updating it.

NSEStalker lets you download data from three different Sources - * NSE Official Website (using bhavcopies) * Yahoo Finance * IB TWS API (Interactive Brokers) Data integrity checking tools are included, which can also check for conformity between data of same granularity but different sources. Once the data has been downloaded and set up in a database, update tools included in the suite can be used to update the database regularly.

Dependencies

R - RMySql, RUnit g++ IB TWS API
